## Why these tools?

The guides focus on modern, fast, and maintainable tooling:

- **UV**: 10-100x faster than traditional Python tools, with unified dependency and Python version management
- **GitHub Actions**: Native CI/CD platform with excellent ecosystem and tight GitHub integration
- **Semantic commits**: Enables automated versioning and changelog generation while improving code history readability

All guides include real-world examples from the opencitations repositories.
